## üìñ What's Inside?
This project demonstrates:
1. **HTML5 Semantic Tags**: Proper use of `<nav>`, `<main>`, `<section>`, and `<footer>`.
2. **Modern CSS**: Flexbox for the navbar and CSS Grid for the Services cards.
3. **Interactive JavaScript**: A mobile "hamburger" menu that toggles on small screens.
4. **Multi-Page Routing**: Linking multiple `.html` files seamlessly.
5. **CI/CD**: Automatic deployment using GitHub Actions.
